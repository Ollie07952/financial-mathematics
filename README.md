## Overview

Within this repository are Python scripts, Jupyter notebooks, and .pdf and LaTeX typeset reports, covering three main areas of financial modelling:

* Derivatives Pricing,
* Modern Portfolio Theory, and
* Time Series Analysis.

All material within this repository is educational and expository in nature, prioritising understanding and reproducibility over deployment-grade implementation. Longer form projects are dedicated their own subdirectories for ease of navigation, but are not necessarily self-contained (i.e. may depend on .py files in the parent directory).



### Disclaimer

All materials herein are intended solely for educational purposes and not financial advice, trading, or decision making in any capacity.
