# Qualitative Portfolio Optimisation



### Introduction

This project explores how techniques from portfolio optimisation theory (similar to those found in Portfolio\_Optimisation\_Theory.ipynb from the parent folder) can be applied to non-standard qualitative data sets. Both the .pdf report and Jupyter notebook detail a framework that takes qualitative relationships and maps them to quantitative outputs to aid resource management decisions.



### Project Structure

* *Qualitative\_Portfolio\_Optimisation.ipynb* — An interactive Jupyter notebook that walks through the construction of the framework. The model offers a high degree of customisability; by modifying the parameters one can apply the model to almost any set of qualitative relations.
* *Portfolio\_Optimisation\_for\_Qualitative\_Data.pdf* — A .pdf copy of the above Jupyter notebook with example parameters and code blocks redacted.
